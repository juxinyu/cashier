Mobile Banking BCA
Login into application BCA Mobile.
Select "m-BCA".
Enter your password m-BCA.
Select "BCA Virtual Account".
Enter BCA Virtual Account Number.
Enter your transaction password m-BCA.
Payment completed.
Internet Banking BCA
Login into KlikBCA Individual.
Enter your User ID dan PIN.
Select "Transfer Dana".
Select transfer to BCA Virtual Account.
Enter BCA Virtual Account Number.
Enter total amount you need to pay.
Validation of payments.
Payment completed.
ATM Bersama
Select "Transaksi L ainnya" at home menu.
Select "Transfer".
Select "Rek Bank Lain".
Blank the "nomor referensi" and click "Benar".
Total amount you need to pay and receive account number will be displayed . Check your payment details to see if it was right select "Benar".
Payment complete.
